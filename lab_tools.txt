## Visual/Animation Effects for Binary File Format

### 25az.bin File Format

**Note:** The 25az.bin format does not have publicly documented specifications. Based on general binary animation formats, possible approaches:

**Potential Structure:**
- Header section with metadata
- Frame data (if animation)
- Color palette
- Texture/sprite data

**Visualization Methods:**
1. Grayscale mapping (byte value to brightness)
2. Histogram visualization
3. Entropy analysis
4. Section mapping (for structured binary)

**Related Tools:**
- bin-graph: Binary visualization tool (https://github.com/8dcc/bin-graph)
- binvis.io: Online binary visualization
- QuickBMS: Binary file extraction

**Animation Frame Extraction:**
- If format contains frame data: sequential extraction
- Delta compression detection (ANIM format methods)
- Palette-based rendering

**Color Palette Generation:**
- Analyze byte distribution
- Map to chemical element colors for thematic effect
- Gradient-based state visualization

## Default Lab Layout Positions

### Workspace Zones
1. **Front-Left**: Sample preparation (mortar, balances)
2. **Front-Center**: Main workspace (beakers, flasks)
3. **Front-Right**: Heating area (Bunsen, hot plates)
4. **Side-Wall**: Safety equipment (eyewash, fire extinguisher)
5. **Back-Right**: Fume hood
6. **Measurement Station**: Right side bench
7. **Storage**: Cabinets along back wall

### Equipment Hierarchy (Z-order)
- Layer 0: Floor markers, shadow
- Layer 1: Large equipment (fume hood, benches)
- Layer 2: Glassware, small equipment
- Layer 3: Active reactions, flames
- Layer 4: Labels, UI elements
